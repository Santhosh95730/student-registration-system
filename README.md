# Student Registration System

## 📌 Project Description

The Student Registration System is a full-stack web application developed to manage student registrations for college events.

Students can register for college events by providing their details, and organizers can view the registered participant information.

---

## 🚀 Features

- Student registration
- College event registration
- View registered students
- Store student details in a database
- REST API backend
- User-friendly interface
